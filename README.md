# 🧑‍🏫 Asesorías UTCAM

Aplicación web creada con **React** para gestionar y consultar asesorías académicas por grupo, materia, docente y carrera en la Universidad Tecnológica de Campeche (UTCAM).

---

## 📌 Funcionalidades

- Visualización de asesorías por materia, grupo y maestro.
- Búsqueda por texto libre (materia, docente, grupo).
- Filtros por grupo o materia.
- Listado de carreras disponibles en UTCAM (TSU e Ingenierías).
- Navegación rápida entre secciones.

---

## 🛠️ Tecnologías utilizadas

- [React](https://react.dev/)
- [Bootstrap 5](https://getbootstrap.com/)
- [React Router DOM](https://reactrouter.com/en/main)
